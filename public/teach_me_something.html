How to setup your own Discord Bot

These steps were designed based on my own experience and setup. There are various other ways to setup your very own bot, but this is the one I like the most.

STEP 1 : Set up a Discord Account

Obviously, you will need a Discord account and your very own server to use and test your bot. If you haven't created an account yet go to Discord.com and create one. If you do have one, log into it and open up a server that you want to use. Also make sure you're an Admin within this server for easier bot to server integration.

You will also need an integrated development environment, I suggest downloading Visual Studio Code to edit your code with.

STEP 2 : Download Node.JS

Node.js is an open source javascript runtime environment that you will need to run your javascript code. You can download it at Nodejs.org and install it on your operating system. You can test if you successfully downloaded it by typing the following separately in your command line terminal :

npm -v
node -v

These should return the version of your installed npm (Javascript Package Manager) and node (Javascript Runtime Evironment).

STEP 3 : Get your Discord application token

The application token acts almost like an API key, a requirement needed for your code to communicate securely and properly with Discord's servers.

To get started go to discordapp.com/developers/applications/me, hit New Application. Give it an appropriate name, then click the button marked Save Changes. On the right-hand menu, click Bot. Once in the menu click Add Bot.

In the box marked App Bot User, look for the phrase Token: Click to Reveal. Click it, and copy the text that appears. Also remember to not share with anyone; the token allows whoever posses it to take full control of your bot all under your account's name.

STEP 4 : Inviting your Discord Bot

Click on your bot’s page. Go to the OAuth2 tab. Tick the bot checkbox under scopes. Tick the permissions required for your bot to function under Bot Permissions. Now the resulting URL can be used to add your bot to a server. Copy and paste the URL into your browser, choose a server to invite the bot to, and click “Authorize”.
